This is not my project!

Because Heroku is not free anymore and other alternative offers poor performance and will most likely turn off my app because there won't be any visitors for some time, I decided to make this warning page. To check the project, just clone my repo and run it locally. It uses mongoose, instead of the cloud mongodb.

Git repo link:

git clone https://github.com/Stefan0712/inventory-app.git

Steps to run the app:

1. Open the terminal/git bash in the folder where you want the repo to clone
2. Paste the git repo link and press enter
3. Enter "npm i" and wait for it to install the dependencies
4. Enter "node app.js" and then go to http://localhost:3000/
